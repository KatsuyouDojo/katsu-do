# Japanese Verbs Web App

This is a lightweight web app to practice Japanese verbs. It works **offline** and is built with Flutter Web.

## Features
- Shows verb forms (base, polite, negative, te-form, ta-form, etc.)
- Works on desktop and mobile browsers
- Offline support once the files are downloaded
- No server or internet connection required
- Simple, clean interface

## How to use
1. Open the `index.html` file in your browser.
2. You can also copy the entire `build/web` folder to your phone or tablet.
3. Works offline without any setup.
